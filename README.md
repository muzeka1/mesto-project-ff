# Mesto
Проект Mesto - сайт для обмена фотографиями
  
## Как пользоваться?
Чтобы поделиться фотографией и изменить ваш аватар, необходима ссылка на картинку.  
Вы можете изменить имя пользователя и добавить описание на аккаунт.
